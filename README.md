# A4Motos 🏍️

**Proyecto formativo — Diseño Frontend Web**

Sitio web de catálogo y venta de motos. Desarrollado con HTML5, CSS3 y JavaScript puro (sin frameworks, sin base de datos).

## 📁 Estructura del Proyecto

```
a4motos/
├── index.html          → Página principal (Hero, destacados, categorías)
├── css/
│   └── styles.css      → Estilos globales con variables CSS, responsive
├── js/
│   └── main.js         → Interactividad: menú, animaciones, formulario
└── pages/
    ├── catalogo.html   → Catálogo con filtros (categoría, marca, precio)
    ├── servicios.html  → Servicios ofrecidos
    └── contacto.html   → Formulario de contacto e información
```

## 🛠️ Tecnologías Utilizadas

- **HTML5** — estructura semántica
- **CSS3** — variables CSS, Flexbox, Grid, animaciones, responsive design
- **JavaScript** — menú burger, Intersection Observer (fade-in), formulario
- **Google Fonts** — Bebas Neue + DM Sans
- **Unsplash** — imágenes de motos (placeholder)

## 🎨 Características de Diseño

- Tema oscuro profesional (negro + rojo)
- Tipografía display (Bebas Neue) + cuerpo (DM Sans)
- Animaciones de entrada (fade-in al hacer scroll)
- Diseño completamente responsivo (mobile, tablet, desktop)
- Navegación fija con menú burger en móvil
- Filtros de catálogo funcionales
- Formulario de contacto con validación

## 📄 Páginas

| Página | Descripción |
|--------|-------------|
| `index.html` | Landing page con hero, stats, motos destacadas y categorías |
| `catalogo.html` | Grid de motos con panel de filtros laterales |
| `servicios.html` | Tarjetas de todos los servicios ofrecidos |
| `contacto.html` | Formulario + información de contacto |

## 🚀 Cómo ejecutar

Abrir `index.html` directamente en el navegador. No requiere servidor ni instalaciones.
